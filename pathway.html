SIGNALING PATHWAY

IL-4 / IL-13 → JAK → STAT6 通路全景

从细胞因子结合到基因表达，STAT6 处于 Th2 型免疫的中枢位置。

Ⅰ 型与 Ⅱ 型 IL-4 受体

Type I 受体 主要在造血细胞

组成：IL-4Rα + γc(common gamma chain)
激酶：JAK1 (IL-4Rα) + JAK3 (γc)
配体：IL-4
主要细胞：T 细胞、B 细胞、肥大细胞、嗜碱粒细胞。

Type II 受体 主要在非造血细胞

组成：IL-4Rα + IL-13Rα1
激酶：JAK1/JAK2 + TYK2
配体：IL-4 和 IL-13
主要细胞：上皮、内皮、平滑肌、成纤维细胞、单核-巨噬细胞。

注意：IL-13Rα2 是高亲和力"诱饵受体"，胞内域短，通常不激活 STAT6，但可通过 AP-1 通路调控 TGF-β。

正/负反馈调控

正反馈

IL-4Rα: STAT6 上调 IL-4Rα 表达，放大 IL-4 信号。
GATA3: Th2 分化后自我维持并促进 IL-4 分泌。

负反馈

SOCS-1/3: 抑制 JAK 激酶活性。
PIAS1: 通过 SUMO 化抑制 STAT6 DNA 结合。
TC-PTP (PTPN2), SHP-1/2: 去磷酸化 STAT6。

与其他通路的 crosstalk

PI3K-AKT: IL-4 也激活 IRS2-PI3K，促进细胞生存增殖。
MAPK: IL-13 → TYK2/JAK2 → ERK 参与上皮修复。
STAT1 拮抗：IFN-γ / STAT1 通路抑制 Th2 分化。

翻译后修饰

磷酸化：Y641（激活）、S756 等。
乙酰化：p300 介导，增强 DNA 结合。
SUMO 化：PIAS 介导，抑制核内活性。
